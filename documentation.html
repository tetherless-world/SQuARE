Ontology Specification Draft

Abstract

The validation ontology include example test cases for each OWL DL axiom. It leverages the Semanticscience Integrated Ontology for existing rule examples, and incorporates its own classes when necessary. For examples that result in inconsistencies, the inconsistencies are commented out in order for the ontology to load properly. The user is recommended to manually uncomment inconsistencies if they wish to test the corresponding axiom.
